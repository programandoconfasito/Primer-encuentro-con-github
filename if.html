Condicionales

Una sentencia condicional es un conjunto de comandos que se ejecutan si una condición es verdadadera
Javascript soporta dos sentencia Condicionales: if...else y switch
Se utiliza la sentencia if para comprobar si la condición lógica es verdadera. Se utiliza la opción esle para ejecutar una sentencia si la condición es falsa.
